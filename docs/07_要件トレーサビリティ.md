## 5. 未確定値の扱い

未確定値は要件漏れではなくM0F成果物である。M0F `GO` 時に本表へ次を追記する。

- SupportProfile ID/hashと対象test generator版。
- ToleranceProfile ID/hash。
- PathCertificate、NoSolutionWitness、canonicalizationのschema ID。
- `runtime-limits-v1.json`の版。
- 基準端末、ブラウザ版、benchmark run ID。

これらが未確定のままM1を開始してはならない。
